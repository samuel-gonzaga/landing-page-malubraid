# Malu Braid — Landing Page

**URL:** [https://malu-braid.landingpro.app.br/](https://malu-braid.landingpro.app.br/)  
**Tipo:** Landing page institucional  
**Finalidade:** Apresentação de serviços de trançamento capilar e captação de leads via WhatsApp.

---

## 📋 Sobre o Projeto

Landing page profissional para divulgação do serviço de tranças **Malu Braid** (Sorocaba/SP). O site foi construído com **WordPress** utilizando o page builder **Elementor**, e está hospedado na plataforma **Hostinger**.

O objetivo principal da página é apresentar o portfólio de trabalhos, listar os serviços oferecidos e direcionar potenciais clientes ao contato direto via WhatsApp.

## 🧩 Funcionalidades da Página

- **Hero section** com chamada principal e CTA para agendamento

## 🧠 Contexto Profissional

Este é **1 de apenas 2 projetos** que desenvolvi com WordPress/Elementor.

### Por que migrei de stack tão rapidamente?

| Problema identificado | Solução adotada |
|---|---|
| Performance mobile inconsistente | Next.js + Tailwind (95+ PSI nativo) |
| Código não versionável (Elementor salva no banco) | Stack 100% com Git |
| Manutenção depende do painel WordPress | Deploy automatizado (Vercel) |
| Limitação em componentes reutilizáveis | React + TypeScript |

### Stack atual

- **Next.js**
- **Tailwind CSS** + shadcn/ui
- **TypeScript**
